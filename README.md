# Actividad-en-clase-Semana-8
Mateo Tipan, Bruno Moreno, Jose Freire y Luis Vásquez

Nuestro programa permite iniciar sesión y luego gestionar información médica a través de diferentes pantallas organizadas con layouts simples y ordenados. Muestra el formulario de inicio de sesión, permite ingresar y validar datos de pacientes, guarda y actualiza citas, gestiona los registros del historial clínico, agregar medicamentos y ajustar dosis. También maneja la navegación entre cada módulo, actualiza automáticamente la interfaz cuando se agrega o modifica la información y conecta los botones, cuadros de texto y listas con la lógica que procesa los datos. Todas las pantallas utilizan principalmente un layout vertical que organiza los campos, botones y listas uno debajo del otro, manteniendo una estructura limpia y fácil de usar. Este layout permite que cada módulo se vea uniforme, alineado y adaptable al contenido que el usuario va ingresando.
